Project README

This project was made using the MERN Stack, which includes:

  • MongoDB for Database
  • Express and Node.js for Backend
  • React (React + Vite) for Frontend

Project Structure

The project is organized into two main directories:

  • client: Contains the frontend files.
    • README.md
  • server: Contains the backend files.
    • README.md

Frontend (client)

The frontend is built using React with Vite, which provides a fast and optimized development experience.

Backend (server)

The backend is built using Node.js and Express, providing a robust and scalable API server.

Getting Started

Prerequisites

Before you begin, ensure you have met the following requirements:

  • Node.js and npm installed on your machine
  • MongoDB installed and running

Installation

To install the project, follow these steps:

Frontend


cd client
npm install
npm run dev

Backend


cd server
npm install
npm start

Usage

To start the development server, run the following commands in separate terminals for the frontend and backend:


# Frontend
cd client
npm run dev

# Backend
cd server
npm start
